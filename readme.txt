ACERCA DEL PROYECTO

Para realizar mi sitio web me basé en una fanpage de un equipo de fútbol.
En este caso elegí el Chelsea Football Club.
Pero se podría aplicar tranquilamente para cualquier otra institución deportiva.
La finalidad del sitio es meramente informativa.


CONSIDERACIONES

• La mayoría del maquetado lo hice con grids para trabajar manualmente e implementar lo visto en el curso,
  tenía mucho bootstrap pero lo fui reemplazando. Lo utilizo solo para algunos detalles y para un modal/formulario.

• Al hacer test con Lighthouse o GTmetrix tirá como resultado baja performance debido al peso de las imágenes.
  Si bien las comprimí, las quise dejar en buena calidad para la presentación del trabajo final.
  Tranquilamente se pueden comprimir un poco más pero ya empiezan a perder calidad.

• Utilicé 2 media querys, uno destinado a tablets y otro a mobile.
  Prioricé el diseño responsive pero el mejor resultado es en desktop.

• Utilicé un solo HTML para trabajar con sections y el navbar solo apunta a dichos sections.
  Pero tranquilamente se podría adaptar a otros HTML en caso de tener una web con más contenido.

• Todas las imágenes estan editadas 100% por mí con Photoshop.
